HOW TO INSTALL PLUGINS ON DISCORD


Discord is a good communication application, created specifically for game communication, you can customize the server in Discord.

But Discord cannot customize the application itself, so if you want to customize the application itself, you must use the help of other tools. One of the best tools is Better Discord.

Step 1

First, click me to open the Better Discord website and download Better Discord.

Step 2

After you have downloaded it, open the application and you will see the following screen as shown below.

Click "I to accept the license agreement", then click Next.

Click "Install BetterDiscord", and click Next.

Select your Discord version, then click Install.

When you see the picture below, it means you have successfully installed Better Discord.

When you open Discord, you should see the screen as shown below.

If you don't see this screen, please go to Discord settings to check if Better Discord is installed successfully.
