HOW TO INSTALL PLUGINS ON DISCORD


Discord is a good communication application, created specifically for game communication, you can customize the server in Discord.

But Discord cannot customize the application itself, so if you want to customize the application itself, you must use the help of other tools. One of the best tools is Better Discord.

Download BetterDiscord

First, click me to open the Better Discord website and download Better Discord.

Install BetterDiscord

After you have downloaded it, open the application and you will see the following screen as shown below.

Click "I to accept the license agreement", then click Next.

Click "Install BetterDiscord", and click Next.

Select your Discord version, then click Install.

When you see the picture below, it means you have successfully installed Better Discord.

When you open Discord, you should see the screen as shown below.

If you don't see this screen, please go to Discord settings to check if Better Discord is installed successfully.

If you see the screen as shown below, you no longer need to worry about anything, because you have successfully installed Better Discord.

Install Plugin

Now, you can install the plugin on Discord! Click me to open Better Discord Plugin website.

Download the plugin you want to install.

Now open your Discord settings, click the plugin option, and click "Open plugin Folder".

Put the plugin you downloaded into the Discord plugin folder you just opened.

Open Discord, you will see Discord has successfully loaded the plugin.

Some plugins will require you to install other plugins before they can be turned on.

After downloading and opening other plugins, you can see that the plug-ins have taken enable successfully.

You have successfully installed the plugin!

PluginRepo

If you find it troublesome to install plugins this way, you can install PluginRepo first.

What is PluginRepo?

PluginRepo is a plugin that allows you to download plugins directly in Discord. This plugin eliminates the tedious steps of installing plugins and can be installed directly after downloading.

Warning

Some plugins may violate Discord's terms of service and may be blocked after installation. Please consider carefully before installation.
For detailed terms of service, please click me.

What is BetterDiscord?

BetterDiscord is a tool for installing plugins on Discord. This tool can modify Discord's local files and change them to what you want.

BetterDiscord can customize CSS, install plugins and themes, and add some tools that Discord doesn't have. Be careful! Installing too many plugins or themes may increase CPU usage.

In short, BetterDiscord is a very good tool, but unfortunately, IOS and Android OS have not yet been launched.
